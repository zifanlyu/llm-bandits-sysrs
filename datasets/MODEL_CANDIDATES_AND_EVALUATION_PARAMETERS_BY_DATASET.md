| Task | Metric | Notes |
|------|--------|-------|

| math | math_equiv_chain_of_thought | Mathematical equivalence |
| gsm | final_number_exact_match | Final number extraction |
| legalbench | quasi_exact_match | 5 legal subsets |
| med_qa | quasi_exact_match | Medical domain |
| narrative_qa | f1_score | Reading comprehension |
| natural_qa | f1_score | 2 modes: closedbook, openbook |
| wmt_14 | bleu_4 | Translation: 5 language pairs |

- Inherited from HELM evaluation protocol (refer to official HELM documentation for complete details on temperature, top-p, max_tokens, etc.)
